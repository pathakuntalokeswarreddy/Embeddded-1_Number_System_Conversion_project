# Number System Conversions in C

This project implements various Number System Conversions using the C programming language. The program is designed to accurately convert numbers between Binary, Decimal, Octal, and Hexadecimal formats while handling all possible edge cases and test scenarios.

## Features

* Cross conversions between Binary, Decimal, octal and Hexa-Decimal
* Input validation for invalid digits and formats
* Handles positive, negative, and zero values
* Supports large numeric inputs within valid range
* Proper error handling and user-friendly output

## Technologies Used

* C Programming
* Standard C Libraries

## Test Cases Covered

* Valid numeric inputs
* Invalid characters in number systems
* Leading zeros
* Maximum and minimum range values
* Zero and negative numbers
* Mixed-case hexadecimal inputs
* Boundary condition testing
